# Elite Realty — Real Estate Landing Page Demo

A modern real estate landing page demo built with **Next.js** and **Tailwind CSS**.  
Elite Realty features elegant layouts, interactive sections, and lead generation tools tailored for real estate businesses.  
Designed for performance, accessibility, and clarity to help showcase properties and attract clients.

---

## Features

- Elegant property showcase with responsive image galleries
- Lead generation forms for inquiries and client conversion
- Responsive layouts optimized for mobile and desktop
- Interactive sections with smooth animations
- Accessibility‑first design with SEO‑friendly structure

---

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [Framer Motion](https://www.framer.com/motion/)
- [Supabase](https://supabase.com/)
